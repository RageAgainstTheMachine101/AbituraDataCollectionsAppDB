# Introduction

This project aims to manage and interact with a database efficiently using Python. Here is a step-by-step guide to help
you set up and work with this database project.

# Prerequisites

Ensure that you have the following installed:

- Python 3.11^
- pip

# Setup

## 1. Clone the Repository

First, clone the repository to your local machine.

```bash
git clone https://github.com/RageAgainstTheMachine101/AbituraDataCollectionsAppDB.git
```

## 2. Navigate to the Project Directory

```bash
cd AbituraDataCollectionsAppDB
```

## 3. Install Dependencies

Install the required Python packages using pip.

```bash
pip install -r requirements.txt
```

## 4. Insert Mock Data

```bash
python [generate_mock_data.py](scripts/generate_mock_data.py)
```

## Generate Tables Relations' Visualization

```bash
python [generate_erd.py](scripts/generate_erd.py)
```
